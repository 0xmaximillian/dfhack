dfhack by peterix and numerous contributors

DFHack is a Dwarf Fortress memory access library, distributed with scripts and plugins implementing a wide variety of useful functions and tools.
Peterix started the project but has become relatively inactive. It's open source so anyone can add changes and make their own releases.
